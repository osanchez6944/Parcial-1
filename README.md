# Parcial 1 - Clasificador de Flores 🌸

Este proyecto utiliza una red neuronal convolucional (CNN) para clasificar 5 tipos de flores.

## 🧠 Modelo Entrenado
Debido al tamaño del archivo (.h5), el modelo se encuentra alojado en Google Drive:
* [👉 Descargar modelo_flores_v2.h5 aquí](PEGA_AQUÍ_TU_LINK_DE_DRIVE)

## 🚀 Instrucciones
1. Descargar el archivo `.h5` y colocarlo en la misma carpeta que `app.py`.
2. Ejecutar `pip install streamlit tensorflow pillow`.
3. Correr la app con `python -m streamlit run app.py`.
